Основы сетевых протоколов

Сетевой протокол
Набор правил, определяющих формат и порядок обмена данными между устройствами в сети.
Полезно знать
Протоколы обеспечивают совместимость оборудования разных производителей и стабильную работу интернета.

TCP/IP

TCP/IP
Базовый стек протоколов интернета, включающий протоколы транспортного (TCP) и сетевого (IP) уровней.
Полезно знать
TCP гарантирует доставку данных, в то время как IP отвечает за маршрутизацию пакетов между узлами сети.

HTTP

HTTP
HyperText Transfer Protocol — протокол прикладного уровня для передачи гипертекстовых документов.
